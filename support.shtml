Support

The following can be used to obtain support for SNMPTT. If you email me direct with a question, I will try to answer it but I would prefer it if the following are tried first.

To protect yourself when posting snmptt.debug logs, modify all ip addresses and all SNMP community names before posting

  • GitHub Issues
  • GitHub Discussions

The following support options are still available for review, but are no longer active:
  • Browse and search SNMPTT-Users mail list archive
  • Browse and search SNMPTT-Developer mail list archive
  • Sourceforge Bug Database
  • Sourceforge Patch Database

Reporting Bugs

Bugs should be reported using the GitHub Issues.

Submitting patches

Patches should be submitted as a pull request or in GitHub Issues.
